## US-PROF-001: Edit Personal Information

**Priority:** P0  
**Role:** Buyer, Agent  
**Traces to:** FR-PROF-001

**As a** user  
**I want** to edit my name, phone number, and profile photo  
**So that** agents and the platform have accurate contact details

---

## US-PROF-002: Set Language Preference

**Priority:** P0  
**Role:** All authenticated users  
**Traces to:** FR-PROF-002, NFR-UX-002

**As a** user  
**I want** to choose Arabic or English as my app language  
**So that** the entire app displays in my preferred language without re-login

---

## US-PROF-003: Save Favorite Properties

**Priority:** P0  
**Role:** Buyer, Agent  
**Traces to:** FR-PROF-003, FR-PROF-004

**As a** user  
**I want** to save properties to favorites and view my favorites list  
**So that** I can quickly return to listings I am considering

---

## US-PROF-004: Set Search Preferences

**Priority:** P0  
**Role:** Buyer, Agent  
**Traces to:** FR-PROF-005

**As a** user  
**I want** to save my budget range, preferred areas, property type, and rent/buy preference  
**So that** search and recommendations default to what I am looking for

---

## US-PROF-005: Manage Notification Preferences

**Priority:** P0  
**Role:** Buyer, Agent  
**Traces to:** FR-PROF-006, FR-NOTIF-003

**As a** user  
**I want** to control push and email notifications by event type  
**So that** I receive only the updates I care about

---

## US-PROF-006: Set Default AI Agent

**Priority:** P0  
**Role:** Buyer, Agent  
**Traces to:** FR-PROF-007

**As a** user  
**I want** to choose my default AI assistant in profile settings  
**So that** new chat sessions start with my preferred agent

---

## US-PROF-007: Agent Public Profile

**Priority:** P0  
**Role:** Agent  
**Traces to:** FR-PROF-008

**As a** real estate agent  
**I want** to maintain a public profile with bio, photo, phone, and service areas  
**So that** buyers trust me when booking viewings

---

## US-PROF-008: Delete Account

**Priority:** P0  
**Role:** All authenticated users  
**Traces to:** FR-PROF-009, NFR-COMP-003

**As a** user  
**I want** to permanently delete my account and personal data  
**So that** I exercise my right to erasure under Egypt PDPL

---

## US-PROF-009: Export Personal Data

**Priority:** P1  
**Role:** All authenticated users  
**Traces to:** FR-PROF-010, NFR-COMP-004

**As a** user  
**I want** to download a copy of my personal data in JSON format  
**So that** I can exercise my data portability rights

---

## US-PROF-010: Bilingual Notifications

**Priority:** P1  
**Role:** All authenticated users  
**Traces to:** FR-NOTIF-004

**As a** user  
**I want** notifications sent in my selected language  
**So that** push and email alerts are readable without switching language

---

## US-PROF-011: View Agent Profile as Buyer

**Priority:** P2  
**Role:** Buyer  
**Traces to:** FR-PROF-008

**As a** buyer  
**I want** to view an agent's profile when booking a viewing  
**So that** I know who I will meet at the property
